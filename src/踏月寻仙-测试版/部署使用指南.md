# 踏月寻仙 - 完整部署使用指南

## 📦 第一步：构建项目

在项目根目录运行：

```bash
pnpm build
```

构建完成后，会在 `dist/美化状态栏/` 生成以下文件：
- `index.html` - 状态栏界面
- `脚本/变量结构/index.js` - 变量结构注册脚本

## 🚀 第二步：在酒馆助手中部署

### 2.1 上传文件到 GitHub

1. 将 `dist/美化状态栏/` 文件夹上传到你的 GitHub 仓库
2. 获取 jsDelivr CDN 链接，格式为：
   ```
   https://cdn.jsdelivr.net/gh/你的用户名/仓库名@分支/dist/美化状态栏/index.html
   https://cdn.jsdelivr.net/gh/你的用户名/仓库名@分支/dist/美化状态栏/脚本/变量结构/index.js
   ```

### 2.2 在酒馆助手中添加脚本

1. 打开酒馆 (SillyTavern)
2. 点击顶部的 "扩展" 图标
3. 找到 "Tavern Helper (酒馆助手)"
4. 点击 "脚本库" 标签
5. 点击 "添加脚本"
6. 填写信息：
   - **脚本名称**: 踏月寻仙-变量结构
   - **脚本 URL**: 你的 jsDelivr 链接（指向 `index.js`）
   - **自动加载**: ✅ 勾选
7. 保存

### 2.3 在酒馆助手中添加界面

1. 在酒馆助手中点击 "界面库" 标签
2. 点击 "添加界面"
3. 填写信息：
   - **界面名称**: 踏月寻仙-状态栏
   - **界面 URL**: 你的 jsDelivr 链接（指向 `index.html`）
4. 保存

## 🎮 第三步：在角色卡中使用

### 3.1 创建角色卡

在酒馆中创建或编辑角色卡，设置以下内容：

#### 第一条消息（多个开局）

将 `src/踏月寻仙-测试版/第一条消息/` 中的开局文件内容复制到角色卡的"第一条消息"中。

例如 **开局1-凡人逆袭.txt**：
```
{{mvu 世界时钟.纪元=末法时代}}
{{mvu 本尊.体质=凡体}}
{{mvu 本尊.灵根=五行杂灵根}}
{{mvu 本尊.身份.宗门=散修}}
{{mvu 本尊.行踪.当前区域=神州}}

---

【开局：凡人逆袭】

你叫{{user}}，一个来自凡间小村的普通少年...
```

**自定义宗门开局示例** - 剑阁弟子：
```
{{mvu 本尊.体质=先天剑体}}
{{mvu 本尊.灵根=金天灵根}}
{{mvu 本尊.身份.宗门=剑阁}}
{{mvu 本尊.身份.出身=剑修世家}}
{{mvu 本尊.行踪.当前区域=剑门关}}
{{mvu 本尊.功法=万剑归宗诀}}

---

【开局：剑阁真传】

你叫{{user}}，剑阁百年一遇的天才剑修...
```

你可以创建多个"第一条消息"选项，每个对应不同的宗门和开局。

#### 系统提示词（可选）

在系统提示词中添加角色设定、世界观等。

#### 世界书（可选但推荐）

如果需要 AI 自动更新变量，在世界书中添加：

1. **变量更新规则** - 告诉 AI 如何更新变量
2. **变量列表** - 列出所有可用变量
3. **变量输出格式** - 定义 MVU 命令格式

（可以参考 `src/美化状态栏/世界书/变量/` 中的模板）

### 3.2 加载状态栏界面

在角色卡的"第一条消息"末尾或系统提示词中添加：

```
{{run:踏月寻仙-状态栏}}
```

这会在消息楼层中加载状态栏界面。

## 💡 第四步：使用和测试

### 4.1 开始对话

1. 选择角色卡
2. 选择一个开局（如"开局1-凡人逆袭"）
3. 开始对话

状态栏会自动显示，并显示开局中设置的初始值（体质、灵根等）。

### 4.2 测试实时更新

在聊天中输入以下命令测试：

```
/setvar key=本尊.体质 value=太虚龙剑体
```

状态栏应该立即更新显示新的体质！

### 4.3 AI 自动更新变量

如果配置了世界书，AI 可以在回复中使用 MVU 命令更新变量：

```
{{mvu 本尊.修为+50}}
{{mvu 本尊.灵石-10}}
{{mvu 本尊.背包.聚气丹.数量-1}}
```

状态栏会自动反映这些变化。

## 🔧 常用命令

### 手动设置变量

```
# 设置基础属性
/setvar key=本尊.体质 value=太虚龙剑体
/setvar key=本尊.灵根 value=变异雷灵根
/setvar key=本尊.修为 value=50
/setvar key=本尊.等级 value=2

# 设置身份信息
/setvar key=本尊.身份.姓名 value={{user}}
/setvar key=本尊.身份.宗门 value=万法宗

# 添加物品（需要完整对象）
/setvar key=本尊.背包.聚气丹 value={"名称":"聚气丹","描述":"低阶修炼丹药","品阶":"凡品","数量":10}
```

### 查看变量

```
# 查看某个变量
/echo {{getvar::本尊.体质}}
/echo {{getvar::本尊.修为}}

# 查看所有变量（会很长）
/echo {{getvar::本尊}}
```

### 重新加载界面

如果状态栏显示不正常：

```
/run 踏月寻仙-状态栏
```

## 🐛 故障排除

### 状态栏显示默认值，不更新

**原因**：脚本未正确加载或 MVU 未初始化

**解决**：
1. 检查浏览器 Console（F12）是否有 `[踏月寻仙] MVU 变量结构已注册`
2. 确认脚本 URL 正确且可访问
3. 刷新页面重新加载

### 开局初始化命令没有执行

**原因**：格式错误或 MVU 框架未加载

**解决**：
1. 确认使用 `<request:{{mvu ...}}>` 格式
2. 检查所有 `{{mvu}}` 命令语法是否正确
3. 等待几秒让 MVU 框架初始化后再发送第一条消息

### 物品、红颜添加失败

**原因**：字段不完整

**解决**：
确保包含所有必需字段：

```
# 物品必需字段
{名称:'xx',描述:'xx',品阶:'xx',数量:1}

# 红颜/NPC 必需字段
{等级:1,修为:0,灵根:'xx',体质:'xx',功法:'xx',本命兵器:'xx',神通列表:{},灵石:0,已活岁月:16,尝试突破:false,好感度:0,关系:'xx',重要性:'xx'}
```
