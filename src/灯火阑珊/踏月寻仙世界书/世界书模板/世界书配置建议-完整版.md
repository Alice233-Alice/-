# 踏月寻仙世界书配置建议 (完整版)

本文档提供所有世界书模板文件在 SillyTavern 中的推荐配置策略。

---

## 🎯 SillyTavern 世界书配置要素

### 1. 激活方式 (第一列圆点)

- **🔵 蓝灯** - 永久激活 (Constant) - 无论什么情况都会激活
- **🟢 绿灯** - 关键词触发 (Keywords) - 只有提到关键词时才激活

### 2. 插入位置选项

#### ✅ 有深度设置的位置

**只有以下三种支持深度和顺序设置**:

- **@D ⊙ [系统]在深度** - 始终插入
- **@D ⚬ [用户]在深度** - 用户输入时插入
- **@D ⊙ [AI]在深度** - AI回复时插入

#### ❌ 无深度设置的位置

**以下位置只能设置顺序,无深度选项**:

- **⬆️ 角色定义之前** - 在角色卡定义内容之前插入
- **⬇️ 角色定义之后** - 在角色卡定义内容之后插入
- **示例消息(↑EM)前**
- **示例消息(↓EM)后**
- **作者注释之前**
- **作者注释之后**

### 3. 配置组合说明

每个世界书条目需要配置:

1. **激活方式**: 🔵蓝灯(永久) 或 🟢绿灯(关键词)
2. **插入位置**: 选择上述位置之一
3. **深度**: 仅@D系列位置可用 (0-9,越小优先级越高)
4. **顺序**: 所有位置都可设置 (数值越小优先级越高)
5. **关键词**: 仅🟢绿灯需要填写 (逗号分隔)

## 🎮 使用技巧

### 插入位置分层逻辑

```
⬆️ 角色定义之前 (优先级最高,在角色卡内容前)
├─ MVU 只读字段说明 (🔵蓝灯, 顺序1)
└─ Visual Cards 系统 (🔵蓝灯, 顺序2)

⬇️ 角色定义之后 (在角色卡内容后)
└─ 红颜索引 (🔵蓝灯, 顺序10)

@D 深度插入 (动态内容,按深度分层)
├─ 深度 2: 核心内容 (🟢绿灯触发)
│   ├─ 核心红颜 (3个)
│   └─ 宗门势力 (3个)
│
├─ 深度 3: 辅助内容 (🟢绿灯触发)
│   ├─ 重要红颜 (3个)
│   ├─ 地形地貌 (4个)
│   ├─ 城镇据点
│   ├─ 功法神通
│   └─ 兵器法宝
│
└─ 深度 4: 次要内容 (🟢绿灯触发)
    └─ 次要红颜 (2个)
```

**位置选择建议**:
- **⬆️ 角色定义之前**: 系统规则、协议定义等需要最优先读取的内容
- **⬇️ 角色定义之后**: 全局索引、概览信息等补充角色定义的内容
- **@D 深度插入**: 具体的角色、地点、物品等动态触发的详细内容

### Token 预算

- **永久激活** (🔵蓝灯): ~310 token
- **平均触发** (🟢绿灯): ~500-800 token
- **总预算**: ~800-1100 token/次对话

### 蓝灯 vs 绿灯选择

| 特征 | 🔵蓝灯(永久) | 🟢绿灯(关键词) |
|------|-------------|---------------|
| 激活条件 | 始终激活 | 提到关键词时 |
| Token消耗 | 每次都消耗 | 按需消耗 |
| 适用内容 | 系统规则、索引 | 具体角色、地点 |
| 推荐数量 | 2-3个 | 不限 |

## 📝 导入步骤

1. **打开世界书**: 在SillyTavern角色卡中打开世界书
2. **创建条目**: 点击"添加条目"
3. **配置设置**:
   - 第一列: 选择🔵蓝灯或🟢绿灯
   - 插入位置:
     - 系统规则选择 ⬆️ 角色定义之前
     - 索引信息选择 ⬇️ 角色定义之后
     - 动态内容选择 @D ⊙ [AI]在深度
   - 深度: @D系列位置填入数值 (2-4),其他位置留空
   - 顺序: 填入数值
   - 关键词: 绿灯时填入,逗号分隔
4. **粘贴内容**: 从YAML文件复制对应内容
5. **测试验证**: 对话中测试是否正确触发

---

## ⚠️ 重要提示

1. **蓝灯不要过多** - 建议只用于系统规则和索引(2-3个)
2. **绿灯要设关键词** - 关键词要全面,包含角色名、所属、特征
3. **角色定义前后的选择**:
   - ⬆️ 角色定义之前: 系统约束、格式协议等必须最先读取的内容
   - ⬇️ 角色定义之后: 世界观索引、角色列表等补充信息
4. **深度越小越优先** - 仅@D系列位置有深度,数值越小优先级越高
5. **顺序用于细调** - 所有位置都可用顺序区分优先级
6. **定期优化** - 根据实际使用调整关键词、位置和深度
