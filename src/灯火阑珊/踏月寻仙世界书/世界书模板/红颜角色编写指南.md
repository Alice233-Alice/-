# 红颜角色编写指南 - 踏月寻仙

## 🎯 目标

帮助你快速、高效地编写符合"踏月寻仙"世界观的红颜角色，同时保持 Token 优化和系统兼容性。

---

## 📋 红颜角色的核心要素

### 1. 基础信息（必须）
- **姓名**：独特且符合修仙风格
- **境界**：等级 1-36（练气初期到渡劫大圆满）
- **灵根**：如五行、变异、特殊灵根
- **体质**：如凡体、特殊体质
- **年龄**：已活岁月

### 2. 修炼数据（必须）
- **修为值**：当前修为进度
- **功法**：所修功法名称
- **本命兵器**：主要法宝
- **神通列表**：已掌握的神通

### 3. 情感数据（红颜特有）
- **好感度**：-100 到 100
- **关系**：陌生人、认识、好友、挚友、道侣等
- **重要性**：路人、配角、主要角色等

### 4. 性格与背景（世界书描述）
- **性格特点**：3-5个关键词
- **背景故事**：简洁有力的 1-2 段
- **与主角的关系**：相遇经历、情感发展
- **角色定位**：在故事中的作用

## 🎨 编写策略：骨架+血肉法

### 核心理念

**世界书写骨架，AI 生成血肉**

世界书只需提供：
- ✅ 关键信息（灵根、体质、境界）
- ✅ 核心性格（3-5个关键词）
- ✅ 重要经历（1-2段概述）
- ✅ 特殊设定（独特之处）

AI 会根据这些骨架：
- 自动生成细节描写
- 根据场景调整表现
- 创造互动内容

### Token 优化建议

| 内容类型 | Token 预算 | 建议 |
|---------|-----------|------|
| 核心红颜（3-5人） | 150-200/人 | 详细描写 |
| 重要红颜（5-10人） | 80-120/人 | 中等描写 |
| 次要红颜（10+人） | 30-50/人 | 简要描写 |

```yaml
  好感度：60/100（好友 → 暗生情愫）

  外冷内热 | 责任感强 | 不善表达 | 刀子嘴豆腐心 | 暗中关心

  正面形象：白衣清冷，如雪中寒梅
  内心反差：外表高冷，内心细腻温柔

【互动提示】
  - 对话风格：简短、直接、偶尔傲娇
```

## 🎭 角色类型参考

```yaml
性格：熟悉、自然、有默契
关系发展：从青梅竹马到情侣
好感基础：50起步
特点：了解主角过去，有共同回忆
```

### 2. 高冷师姐型
```yaml
性格：冷淡、强大、不易亲近
关系发展：从冷淡到逐渐打开心扉
好感基础：0-10起步
特点：实力强，被主角的某个特质吸引
```

### 3. 温柔师妹型
```yaml
性格：温柔、依赖、需要保护
关系发展：从仰慕到爱恋
好感基础：20-30起步
特点：天赋好但缺乏自信，被主角鼓励
```

### 4. 傲娇大小姐型
```yaml
性格：骄傲、任性、刀子嘴豆腐心
关系发展：从互相看不惯到傲娇喜欢
好感基础：-10到10起步
特点：出身高贵，主角不按常理出牌让她意外
```

### 5. 敌对转化型
```yaml
性格：初期敌对、后期复杂
关系发展：从敌人到理解到暧昧
好感基础：-50到-20起步
特点：立场不同，但被主角人格魅力征服
```

### 6. 神秘强者型
```yaml
性格：神秘、强大、难以捉摸
关系发展：慢热，逐渐展露真实
好感基础：0起步
特点：实力远超主角，被主角潜力吸引
```

## 💡 实用技巧

### 技巧1：性格关键词法

用 3-5 个关键词定义性格，AI 会自动演绎：

```yaml
# 示例组合
组合A：温柔、善良、胆小、依赖 → 需要保护的师妹
组合B：冷淡、强大、责任、孤独 → 高冷强大的师姐
组合C：活泼、好奇、冲动、真诚 → 元气满满的师妹
组合D：骄傲、任性、善良、傲娇 → 大小姐类型
```

### 技巧2：对比设定法

通过对比让角色更立体：

```yaml
外表 vs 内心：
  - 外表冷淡 vs 内心温柔（孟十七）
  - 外表柔弱 vs 内心坚强
  - 外表高贵 vs 内心孤独

实力 vs 性格：
  - 实力强大 vs 性格温柔
  - 实力一般 vs 性格坚毅
  - 天赋异禀 vs 自卑敏感
```

### 技巧3：特殊设定增色

为角色添加独特元素：

```yaml
【身体特征】
  - 特殊体质（会发作/需压制）
  - 封印血脉（未觉醒）
  - 特殊标记（胎记/符文）

【背景秘密】
  - 隐藏身份
  - 悲惨过往
  - 特殊使命

【能力特长】
  - 炼丹/炼器高手
  - 阵法大师
  - 音律神通
```

### 技巧4：关系网设计

不要让红颜孤立存在：

```yaml
孟十七的关系网：
  - 师傅：寒霜峰峰主（养育之恩）
  - 师妹：白小蝶（好友，互相扶持）
  - 对手：他宗天才（竞争关系）
  - 暗恋者：同门师兄（单相思）
```

### 技巧5：好感度分级设计

```yaml
-100 ~ -50: 仇敌（生死之仇）
 -50 ~ -20: 敌对（立场对立）
 -20 ~   0: 陌生/警惕
   0 ~  20: 认识/普通
  20 ~  40: 朋友
  40 ~  60: 好友/暗生好感
  60 ~  80: 暧昧/喜欢
  80 ~ 100: 深爱/道侣

好感度变化触发事件：
  - 30突破：愿意倾诉心事
  - 50突破：会主动关心主角
  - 70突破：表白或被表白
  - 90突破：考虑结为道侣
```

## 📚 完整示例：从零创建红颜

### 第一步：确定角色定位

```
想法：创造一个神秘的丹药师，外表温柔但实力深不可测
定位：重要配角，后期可能发展为主要角色
```

### 第二步：填充基础框架

```yaml
[角色] 苏寒烟

【基础】
  姓名：苏寒烟
  境界：元婴后期（等级16）
  灵根：变异丹灵根
  体质：药神体
  年龄：外表25，实际200岁
  所属：散修，万丹阁供奉

【修炼】
  功法：《百草丹经》（天阶）
  本命兵器：紫金丹炉（灵宝）
  神通：
    - 炼化万物：可炼制任何丹药
    - 药气化形：以药气凝聚战斗
```

### 第三步：添加情感与性格

```yaml
【情感】
  好感度：0（初遇陌生）
  关系：陌生人 → 忘年交 → 红颜知己
  重要性：重要配角

【性格】
  温柔恬静 | 深藏不露 | 看透世事 | 孤独 | 珍惜真诚
```

### 第四步：编写背景故事

```yaml
【背景】
  出身：
    百年前的顶尖丹道天才，因拒绝邪修拉拢被追杀，
    假死遁走，隐姓埋名成为万丹阁供奉。
  
  现状：
    表面是温柔的丹药师，实则实力深不可测。
    已看透名利，只想安静炼丹，寻找真正的道友。
  
【与主角】
  相遇：主角求丹，她被主角真诚打动，破例炼制
  关系：惜才，视主角为忘年交，偶尔指点
  契机：主角无意中发现她的秘密，选择保守，获得信任
```

### 第五步：添加特殊设定

```yaml
【特殊】
  秘密：曾是某大宗门圣女，背负血海深仇
  特长：丹道无双，可炼制失传丹药
  弱点：心结未解，修为停滞
  目标：复仇？还是放下？（剧情可选）
  
【互动】
  - 送珍贵丹药会大幅提升好感
  - 帮助她解开心结可触发特殊剧情
  - 炼丹相关话题可加深交流
```

### 第六步：Token 优化

```yaml
# 优化前：约 250 token
# 优化后：约 120 token

【优化版】
苏寒烟 | 元婴后期(16) | 丹灵根 | 药神体 | 外表25实际200

【修炼】百草丹经(天阶) | 紫金丹炉(灵宝) | 炼化万物、药气化形

【情感】好感0 | 陌生→知己 | 重要配角

【性格】温柔恬静 | 深藏不露 | 孤独 | 珍惜真诚

【背景】
  百年前丹道天才，拒邪修被追杀假死，现为万丹阁供奉。
  实力深不可测，寻找真正道友。

【与主角】求丹相遇，被真诚打动；发现秘密选择保守，获信任

【特殊】曾为大宗圣女，背负血仇；丹道无双；心结未解；可触发复仇线

【图片】normal, smile, sad, serious
```

## 🎨 最后的建议

1. **质量优于数量**：3个有深度的红颜 > 10个平面角色
2. **留白的艺术**：不要写满所有细节，给AI发挥空间
3. **动态发展**：好感度会变，关系会变，性格也可能成长
4. **符合世界观**：境界、功法、神通要符合设定
5. **个性鲜明**：每个角色应该有独特的"记忆点"
