# 踏月寻仙世界书配置建议

本文档提供所有世界书模板文件在 SillyTavern 中的推荐配置策略。

---

## 🎯 SillyTavern 触发策略说明

### 有深度设置的策略 (可设置深度和顺序)

根据你提供的截图,**只有以下三种策略支持深度设置**:

1. **@D ⊙ [系统]在深度** - 始终激活,在指定深度插入
2. **@D ⚬ [用户]在深度** - 用户输入时激活,在指定深度插入  
3. **@D ⊙ [AI]在深度** - AI回复时激活,在指定深度插入

### 无深度设置的策略 (仅可设置顺序)

以下策略**没有深度选项**,只能设置顺序:

4. **角色定义之前** - 插入在角色卡描述之前
5. **角色定义之后** - 插入在角色卡描述之后
6. **示例消息(↑EM)前** - 插入在示例对话之前
7. **示例消息(↓EM)后** - 插入在示例对话之后
8. **作者注释之前** - 插入在作者注释之前
9. **作者注释之后** - 插入在作者注释之后

### 1. MVU 只读字段说明

**条目名称**: `[系统] MVU只读字段说明`

**配置**:
- **触发策略**: @D ⊙ [系统]在深度
- **深度**: 0
- **顺序**: 1
- **触发关键词**: 留空 (系统策略始终激活)

**理由**:
- 系统规则,必须始终激活
- 深度 0 确保最先被 AI 看到
- 顺序 1 最高优先级

## 🎮 实际使用建议

### 深度分层逻辑

```
深度 0: 系统规则 (始终最优先)
├─ MVU 只读字段说明
└─ Visual Cards 系统

深度 1: 索引和概览
└─ 红颜索引

深度 2: 核心内容 (重要且详细)
├─ 核心红颜角色 (3个)
└─ 宗门势力库 (拆分3个)

深度 3: 辅助内容 (补充信息)
├─ 重要红颜角色 (3个)
├─ 地形地貌库 (拆分4个)
├─ 城镇据点库
├─ 功法神通库
└─ 兵器法宝库

深度 4: 次要内容 (可选细节)
└─ 次要红颜角色 (2个)
```

### Token 预算控制

1. **始终激活的条目** (系统策略):
   - MVU 只读字段说明 (~150 token)
   - Visual Cards 系统 (~100 token)
   - 红颜索引 (~60 token)
   - **总计**: ~310 token

2. **动态激活的条目** (AI策略,按需触发):
   - 根据对话内容和关键词自动激活
   - 平均每次对话: 300-800 token

3. **总 Token 预算**:
   - 最小: ~310 token (仅基础系统)
   - 平均: ~800-1100 token (正常对话)
   - 最大: ~2000 token (复杂场景)

### 关键词设置技巧

1. **包含角色名和别名**: 如"敖清霜"和"敖灵儿"
2. **包含所属势力**: 如"万法宗"、"问道峰"
3. **包含关键特征**: 如"冰系"、"玄冰灵体"
4. **避免过于常见的词**: 不要用"的"、"是"这类
5. **用逗号分隔**: `关键词1, 关键词2, 关键词3`

## 📝 导入步骤

1. **在 SillyTavern 中打开角色卡的世界书**

2. **创建新条目并配置**:
   - 点击"添加条目"
   - 粘贴对应 YAML 文件中的内容
   - 选择触发策略 (根据上表)
   - 设置深度值 (如适用)
   - 设置顺序值
   - 填入触发关键词

3. **测试验证**:
   - 在对话中提到相关内容
   - 检查条目是否正确触发
   - 调整关键词以优化触发

---

## ⚠️ 重要注意事项

1. **只有 @D 开头的策略支持深度**
   - @D ⊙ [系统]在深度
   - @D ⚬ [用户]在深度
   - @D ⊙ [AI]在深度

2. **深度值越小优先级越高**
   - 深度 0 的内容会最先被 AI 看到
   - 在 Token 限制下,高深度内容可能被截断

3. **系统策略 vs AI策略**
   - 系统策略: 始终激活,适合规则和索引
   - AI策略: 按关键词触发,适合具体内容
   - 不要过度使用系统策略,会浪费 Token

4. **关键词要全面但不冗余**
   - 测试常用表述是否能触发
   - 避免使用过于宽泛的词
   - 定期根据使用情况优化
