# 随机秘境任务系统 - 使用示例

## 概述
秘境探索系统允许 AI 根据剧情动态生成随机秘境任务，为玩家提供更丰富的游戏体验。

## Schema 结构
```typescript
任务列表: {
  [任务名]: {
    类型: '主线' | '支线' | '每日' | '临危受命' | '秘境探索' | '暂无',
    目标: string,
    状态: '进行中' | '已完成' | '已失败' | '暂无',
    秘境信息?: {
      域?: '天层' | '神州' | '东苍' | '南炎' | '西庚' | '北冥' | '下层' | '四海',
      危?: number, // 0-100
      特?: string, // 秘境特点
      奖?: string[], // 可能的奖励
      限?: string, // 时间限制或其他限制
    }
  }
}
```

## 触发时机
AI 可以在以下情况生成秘境任务：
1. 玩家到达新区域
2. 遇到特定 NPC 提供线索
3. 触发世界事件（天降异象、空间裂隙等）
4. 修为突破后解锁新区域
5. 完成前置任务后开启

## 秘境类型示例

### 1. 古修士洞府
```yaml
探索青云真人洞府:
  类型: 秘境探索
  目标: 探索青云真人遗留洞府，获取传承
  状态: 进行中
  秘境信息:
    域: 神州
    危: 45
    特: 机关重重，需破解阵法；内有筑基期傀儡守护
    奖: [玄阶功法残卷, 筑基丹x3, 中品灵石x100, 储物戒]
    限: 7日内完成，否则洞府自毁
```

### 2. 上古遗迹
```yaml
探索朱雀试炼地:
  类型: 秘境探索
  目标: 通过朱雀氏族遗留的试炼，获得朱雀血脉
  状态: 进行中
  秘境信息:
    域: 南炎
    危: 85
    特: 火焰试炼，需抵御朱雀真火；考验火系感悟
    奖: [朱雀血脉(小成), 朱雀涅槃经(残), 朱雀羽x1, 离火精华x50]
    限: 仅化神境以上可入；失败将受朱雀真火反噬
```

## 难度匹配原则
- **练气期(1-4级)**: 危险度 10-30，低级洞府、药园
- **筑基期(5-8级)**: 危险度 30-50，古修士洞府、小型遗迹
- **金丹期(9-12级)**: 危险度 50-70，宗门秘境、试炼之地
- **元婴期(13-16级)**: 危险度 70-85，上古遗迹、禁地外围
- **化神期(17-20级)**: 危险度 85-95，禁地核心、仙人洞府
- **炼虚以上(21+级)**: 危险度 95-99，绝地、上界碎片

## 奖励设计原则
1. **功法/神通**: 与秘境主题相关，阶位匹配难度
2. **法宝**: 品阶与危险度成正比
3. **材料**: 炼丹/炼器/修炼资源
4. **感悟**: 境界突破契机、特殊体质/血脉
5. **灵石**: 数量与危险度成正比

## AI 生成建议
当 AI 需要生成秘境时，应考虑：
1. **剧情相关性**: 秘境与当前剧情、区域、NPC 有关联
2. **难度合理**: 危险度与玩家境界匹配（±1大境界）
3. **奖励吸引**: 包含玩家当前需要的资源/传承
4. **时间限制**: 增加紧迫感，避免无限拖延
5. **失败后果**: 明确失败代价，增加决策深度
6. **多样性**: 避免重复相似秘境类型

```
  类型: 秘境探索,
  目标: 探索{{user}}在神州发现的神秘洞府,
  状态: 进行中,
  秘境信息: {
    域: 神州,
    危: 40,
    特: 阵法守护需破解;内有筑基傀儡,
    奖: [玄阶功法, 筑基丹x2, 灵石x50],

  }

```

## 注意事项
1. 秘境信息仅在类型为"秘境探索"时需要填写
2. 完成的秘境可以删除或保留作为历史记录
3. 同一秘境不应重复探索（除非有特殊设定）
4. 危险度应与实际遭遇的敌人/机关相符
5. 奖励列表为可能获得的奖励，不是全部都能获得
