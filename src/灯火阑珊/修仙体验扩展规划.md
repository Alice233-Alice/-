# 《灯火阑珊》修仙体验扩展规划

> 本文档用于整理世界观与剧情体验的后续扩展方向，不是需要直接启用的世界书条目。

## 一、核心判断

《灯火阑珊》目前并不缺少宏观设定，已有内容已经覆盖：

- 三界、五域、四海、禁地与天地通道；
- 九大境界、绝对境界压制与高阶神通；
- 正道、魔道、中立、异族等不同道统势力；
- 灵根、寿元、灵石、灵脉、百艺、坊市与因果；
- 偏重忠贞、纯粹、长期陪伴的红颜情感基调；
- 修为、突破、任务、声望、红颜关系、世界时钟等变量支持。

当前最需要补充的不是更多地图、宗门、境界或红颜，而是连接这些设定的“中层体验”：

1. 玩家为什么需要出门历练；
2. 玩家具体怎样修炼与突破；
3. 每次选择必须放弃或承担什么；
4. 一次事件如何产生长期后果；
5. 修行如何改变感情，感情又如何影响修行。

换句话说，已有内容主要回答了“这个世界是什么”，下一步应重点回答“玩家如何生活在这个世界里”。

## 二、建议确立的核心体验循环

后续新增设定与剧情规则，尽量服务于以下循环：

> 遇到瓶颈或欲望 → 查找线索 → 筹集资源与人情 → 出门历练 → 做出取舍 → 承担后果 → 闭关炼化 → 破境或受挫 → 宗门、世界与红颜作出回应

这一循环可以同时承载修炼、冒险、宗门、资源、因果和恋爱，不必把“修仙剧情”与“恋爱剧情”割裂成两套内容。

一次完整的中型剧情，最好至少触及其中三个环节；一次大境界突破，则可以完整走完一轮。

## 三、优先扩展模块

### 1. 大境界质变与破境流程

现有境界设定已经说明了突破后的威能，但还需要补充“怎样成为那种存在”。

每个大境界建议回答以下问题：

- 修为积累之外，还需要满足什么条件；
- 该境界必须悟明什么道理或解决什么心结；
- 需要准备哪些丹药、阵法、天材地宝或特殊环境；
- 突破过程会出现什么肉身、神魂或天地异象；
- 失败会留下什么长期损伤，而非简单回到原状；
- 突破成功后需要多久稳固，又会改变哪些生活方式；
- 师门、仇敌与红颜分别能怎样参与其中。

大境界突破应当是一段剧情，而不只是修为达到阈值后的等级增加。

建议未来拆分条目：

- `[修炼] 大境界质变与破境流程.yaml`
- `[修炼] 突破准备、失败与稳固.yaml`

### 2. 功法、道基与修行代价

功法不应只有名称、品阶和威力，还应代表修士选择的一种生存方式。

每门重要功法可以具备：

- 灵根、体质与环境适性；
- 修炼速度与资源需求；
- 擅长方向与明确短板；
- 对肉身、性格、情绪或道心的长期影响；
- 必须遵守的戒律与走火入魔风险；
- 改修、兼修、散功重修的代价；
- 与其他功法、神通、法宝的组合关系。

应尽量避免无条件的最优功法。选择一条道，就意味着放弃另一部分可能性，或者承担与之对应的代价。

建议未来拆分条目：

- `[修炼] 功法道基与修行代价.yaml`
- `[修炼] 道心、心魔与择道规则.yaml`

### 3. 宗门日常与贡献循环

现有宗门已经具备清晰的道统理念，下一步应让这些理念落实到门人的日常生活中。

可考虑补充：

- 入门考核、根骨测试与拜师仪式；
- 杂役、外门、内门、真传、亲传之间的真实差异；
- 月俸、洞府、丹药、授课与藏经阁权限；
- 贡献点、宗门任务、驻守、巡山与征召；
- 师徒传承、同门竞争、派系分歧与资源分配；
- 小比、大比、论道会、真传试炼与继位之争；
- 门规、惩处、叛门、逐出与追责；
- 宗门覆灭后弟子、灵脉和传承的流散。

每个重要宗门的理念，最好落实为：

> 一项实际好处 + 一项必须遵守的限制 + 一次无法两全的选择

这样宗门不再只是背景阵营，而会成为玩家真实生活和成长的一部分。

建议未来拆分条目：

- `[宗门] 宗门日常与贡献制度.yaml`
- `[宗门] 师徒、同门与传承秩序.yaml`

### 4. 大能约束与护道边界

这是当前最需要优先解决的世界一致性问题。

世界中的高阶修士拥有极强威能，同时部分红颜境界远高于早期玩家。若她们可以随时亲自解决问题，那么灵石、任务、秘境、宗门冲突和低阶敌人都会失去意义。

因此需要建立一套“大能为何不能随便出手”的规则，例如：

- 高阶力量在凡界受到天道、空间或界域压制；
- 本体降临、赐予力量或改写因果都会招致反噬；
- 高阶修士肩负镇守、封印、闭关、伤势或自身劫难；
- 一位大能出手会引来敌对大能，使局势直接升级；
- 可以阻止玩家被越境抹杀，却不能代替玩家争夺同层机缘；
- 越高阶的馈赠，玩家承担的因果、觊觎和炼化难度越大；
- 红颜可以护道、教导、留后手，但不能替玩家完成悟道与破境。

这不是削弱红颜，而是确保她们的强大不会让玩家的修行失去意义。

建议未来拆分条目：

- `[世界] 大能约束与界域压制.yaml`
- `[道侣] 护道、馈赠与代为出手边界.yaml`

### 5. 因果、伤势与长期后果

修仙故事中的失败不一定意味着死亡，更适合转化为后续剧情。

可以建立以下长期后果：

- 道基暗伤、神魂裂痕或经脉隐患；
- 心魔种子、执念、恐惧与错误感悟；
- 欠下护道之恩、救命之恩或宗门人情；
- 与某个家族、宗门或传承结下仇怨；
- 错过秘境、拍卖会、宗门大比等时间窗口；
- 为疗伤消耗寿元、资源或闭关岁月；
- 获得宝物的同时继承原主因果；
- 为兑现誓言而承担未来义务。

失败的设计原则应是：

> 不轻易终止故事，而是改变故事接下来要面对的问题。

建议未来拆分条目：

- `[因果] 恩怨、誓约与因果债.yaml`
- `[修炼] 伤势、道基与失败后果.yaml`

### 6. 世界时钟与长期事件链

随机事件不应只在当前回合制造一段插曲，也可以成为会自行演化的世界事件。

建议采用以下结构：

> 征兆出现 → 玩家是否介入 → 各方势力行动 → 事件升级或转向 → 结果扩散 → 数月或数年后的回响

例如玩家没有理会一次地脉异常，异常也不会凭空消失，而可能逐渐演化为灵矿现世、妖兽迁徙、村镇灾害或宗门争夺。

可以加入的长期时间节点包括：

- 秘境周期性开启；
- 宗门大比与收徒大典；
- 坊市拍卖会和大型交易会；
- 兽潮、灵潮、地火与天象周期；
- 高阶修士出关、坐化或渡劫；
- 王朝更替、宗门战争与灵脉枯竭；
- 红颜的约定、闭关期限与个人任务。

建议未来拆分条目：

- `[事件] 世界时钟与长期事件链.yaml`
- `[事件] 地域事件、势力行动与后续回响.yaml`

### 7. 修仙百艺与资源链

当前设定已经列出丹、器、符、阵，但还可以让其中一至两项成为玩家真正参与的成长路线。

不必为所有百艺编写百科全书，只需让关键百艺具备完整过程：

> 获得传承或丹方 → 辨认材料 → 采集或交易 → 寻找炉火与场所 → 炼制并承担失败风险 → 成品自用、出售或赠予 → 引发新的名声与需求

百艺可以承担低烈度的修仙日常，也能自然连接坊市、宗门、资源、红颜和人情往来。

## 四、恋爱与修仙的融合方式

恋爱不应只是修炼剧情之间插入的温存场景，而应直接发生在修行选择之中。

### 1. 关系推进依据

关系阶段不要只依赖好感度数字，还可以依赖共同经历和真实承诺：

- 是否共同经历过危险；
- 是否互相透露过功法与身世秘密；
- 是否在对方最脆弱时护过道；
- 是否完成过一项长期约定；
- 是否愿意共同承担因果；
- 是否真正理解并认可对方所修之道；
- 是否在大道分歧后仍作出了清醒选择。

### 2. 可与修行结合的情感场景

- 为对方护法、挡劫或守候闭关；
- 共同争夺一份只能由一人使用的破境资源；
- 合炼本命法宝、阵盘或信物；
- 共同参悟功法，创造合击或互补神通；
- 一方境界提升后寿元、身份与责任发生改变；
- 一方想保护，另一方要求保留独自面对危险的尊严；
- 在使用禁术、牺牲他人或承担业障的问题上产生分歧；
- 闭关数十年或相隔两界之前作出约定；
- 进入对方心劫、记忆、领域或体内世界；
- 面对飞升、镇守、宗门责任与长久离别。

### 3. 红颜独立性的保留

忠贞不等于顺从，纯爱也不等于没有分歧。

每位红颜最好都明确拥有：

- 自己正在追求的道；
- 不愿退让的底线；
- 希望玩家理解或帮助的事情；
- 即使相爱也不会答应的要求；
- 为这段感情愿意付出的代价；
- 与玩家功法、身份或道路之间的互补与冲突；
- 区别于其他红颜的表达爱意方式。

她们可以绝不背叛，却仍然会拒绝、迟疑、坚持己见、暂时离开完成责任。这样的矛盾不会破坏纯爱，反而能让感情具有真实重量。

### 4. 多红颜誓约的一致性

现有设定同时包含“道侣向一人起誓”和“玩家可与数位红颜结缘”，两者需要进一步明确规则，避免模型产生不同解释。

后续需要确定：

- 多人情缘是否受到天道认可；
- 红颜之间是否共同知情并认可；
- 情缘与正式道侣的区别；
- 玩家是否也受到忠贞、责任与誓约约束；
- 多人共誓如何成立，违背后的代价是什么；
- 争宠、陪伴时间和资源分配怎样处理，但不发展为恶意倾轧。

这一部分只需明确世界规则，不需要改变当前偏好的纯爱和多红颜方向。

建议未来拆分条目：

- `[道侣] 情缘阶段与立誓规则.yaml`
- `[道侣] 同修、护道与大道分歧.yaml`
- `[道侣] 多人共誓与彼此相处原则.yaml`

## 五、内容实施顺序

不建议同时扩写全部模块，可按以下阶段推进。

### 第一阶段：先解决核心修仙体验

1. 大境界质变与破境流程；
2. 功法道基与修行代价；
3. 宗门日常与贡献循环；
4. 大能约束与护道边界。

完成这四项后，早期、中期和高阶剧情的基本逻辑便能稳定成立。

### 第二阶段：让选择产生持续影响

1. 因果债与誓约；
2. 伤势、心魔和突破失败；
3. 世界时钟与长期事件链；
4. 势力行动与世界回响。

### 第三阶段：丰富生活感与感情深度

1. 选择一至两种修仙百艺重点展开；
2. 将每位核心红颜的个人道路接入修炼循环；
3. 设计护道、闭关、共炼、共悟与大道分歧事件；
4. 明确多红颜情缘的天道誓约规则。

## 六、暂时不必优先扩充的内容

在上述结构完成之前，以下内容可以暂缓：

- 继续增加新大陆与新禁地；
- 继续增加大量宗门名称；
- 继续增加更高境界和更夸张的威能；
- 继续批量增加红颜角色；
- 编写大量不会进入剧情的丹药、妖兽与法宝百科；
- 为每一项修仙百艺建立过于细碎的制作表。

这些内容可以增加信息量，却不一定增加实际游玩体验。

## 七、新设定自检表

以后准备增加任何设定时，可以先问：

1. 它会促使玩家采取什么行动？
2. 它要求玩家在什么事物之间作出选择？
3. 玩家要为选择承担什么代价？
4. 结果能否持续影响三段剧情以上？
5. 宗门、敌人或世界会怎样回应？
6. 它会怎样改变至少一段人物关系？
7. 它是否会被高阶红颜或现有能力轻易绕过？
8. 它是否与当前境界、地图、因果和誓约规则一致？

如果一个设定只能回答“它很酷”或“它让世界更大”，它更接近背景资料；如果能够回答其中五项以上，它通常已经可以形成真正的剧情体验。

## 八、最终方向

《灯火阑珊》理想的体验不应是“修炼一段，再谈一次恋爱”，而应当是：

> 玩家因为所爱之人重新理解自己的道，又因为选择的道决定怎样去爱；每一次破境、远行、争夺、承诺与离别，都同时推动修为和关系。

当修炼与感情共享同一套选择、代价和后果时，角色卡便会自然呈现出“既有正统修仙感，也有长期恋爱陪伴”的体验。
